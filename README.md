# BTRT

BTRT ("Build the Right Thing") is my collection of advice, guidance and tools for Product Success.

Coming from a strong UX and Product Design background, I've designed apps for startups and internal tools for Enterprise.

But BTRT is more than a Product Design framework I've also run my own startup so it'll cover Business Advice – Proposition Refinement, Feasbility, Research.

Finally, I'm also a full-time dev (Typescript + Effect.ts) so I include patterns for Software design too.

## Commercial

First and foremost, the goal is to earn a passive income. I will contribute more content over time. At first I thought about having a blog, but I don't know what to write about yet.

I do know that 80%—90% of my content will be free. I'll upself the remaining 10%. Customers will be able to buy everything for a single, one-off free.

I'll keep all the content in a private Github Repo. When someone buys the content, I'll either invite them or zip the repo and send them an email.
