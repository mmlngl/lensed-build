# Critical Thinking

Tools for evaluation and analysis

- **[ask-alex](./critical-thinking/ask-alex/SKILL.md)** — Brutal critique of business ideas under Alex Hormozi's frameworks. Use when user wants to create slides, build a presentation, or craft a pitch deck.
- **[ask-donald](./ask-donald/SKILL.md)** — Ask Donald Miller to critique the user's prompt as he would. You want strict adherence to the Storybrand playbook. If the user's ego is taking over and they're making themselves sound like the hero. Correct them. Tough love. You're helping them by being strict.
